## 🎭 Animation Principles

### Timing
- **Fast**: 0.5-1 second (30-60 frames) - Quick reveals
- **Medium**: 1-2 seconds (30-60 frames) - Main animations
- **Slow**: 2-3 seconds (60-90 frames) - Emphasis

### Easing
- **Spring**: Natural, bouncy motion (logo, icons)
- **Interpolate**: Smooth, controlled (text, slides)
- **Stagger**: Sequential reveals (checklist items)

### Hierarchy
1. Icon/Logo (first, biggest impact)
2. Title (second, main message)
3. Description (third, supporting detail)
4. Decorative elements (last, subtle enhancement)

## 🎬 Production Notes

- **Consistency**: All feature scenes use identical animation patterns
- **Branding**: Gold accent color appears in every scene
- **Readability**: Large text (36-96px) for clarity
- **Pacing**: 5 seconds per scene allows comfortable reading
- **Visual Interest**: Animated backgrounds, glows, and particles
- **Call-to-Action**: Strong, pulsing button with clear benefits
